## Infinispan Index Admin Demo

This demo creates a local 2-node cluster with indexed entities to try index administration operations via REST.

#### Requirement 

* sdkman: 

    ```curl -s "https://get.sdkman.io" | bash```
     
* scala

    ```sdk use scala 2.11.12```
    
### Running 

```./run.sh <SERVER_HOME>```


### Stopping

```./kill.sh```
 
### Cleanup

```rm -Rf ~/.indexPerf```
